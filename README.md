# todo-list
A simple application with Redux and React.

## How to install
```shell
$ git clone https://github.com/dongrenguang/todo-list.git
$ cd todo-list
$ npm install
```

## How to build
```shell
$ npm run build
```

## How to run
```shell
$ npm start
```
Then, open your browser and visit [http://localhost:8080](http://localhost:8080).
